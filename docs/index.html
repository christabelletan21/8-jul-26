DevAcademy — Software Development Education Platform

A structured-learning platform for software developers: browse courses, track progress on a personal dashboard, and sign in via GitHub OAuth or a dev-only login. Built as a real, working vertical slice on top of a full monorepo scaffold — not a mockup.

View on GitHub Setup Instructions

Tech stack

  • Next.js (App Router, TypeScript)
  • Express
  • PostgreSQL
  • Prisma
  • NextAuth.js
  • Tailwind CSS
  • npm workspaces monorepo

What's actually working

One full vertical slice is implemented end-to-end, plus the supporting project structure, database schema, and authentication:

  • Browse courses and view a course detail page
  • Mark course progress and see it reflected on your dashboard
  • Authentication via NextAuth (GitHub OAuth or dev-only login)
  • Contact form with a voice confirmation and balloon animation on submit

Live trainings/TMS, code playground, AI tutor, payments, and the community forum appear in the navigation as honest "Coming soon" placeholders — they are not built yet.

Run it locally

Full instructions are in the README, in short:

npm run db:up → copy the .env.example files → npm install → npm run db:migrate && npm run db:seed → npm run dev
This page is a static project overview served via GitHub Pages — the app itself (Next.js + Express + Postgres) requires a server and database, so it runs locally or on your own hosting, not from Pages. See the repository for source and setup.
